# 🐵 Userscripts

My personal collection of [Greasemonkey](https://www.greasespot.net/) / [Tampermonkey](https://www.tampermonkey.net/) userscripts.

## What are userscripts?

Userscripts are small JavaScript programs that run in your browser and modify web pages — automating tasks, fixing annoyances, or adding new features to sites.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) (Chrome/Edge/Firefox) or [Greasemonkey](https://www.greasespot.net/) (Firefox)
2. Click the **Raw** button on any `.user.js` file in this repo
3. Your browser extension will prompt you to install it
